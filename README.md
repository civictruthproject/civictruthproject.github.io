# Civic Truth Project

A platform for citizen advocacy, investigative transparency, and collective action.

## Features

- **Speak Truth**: Investigative reports countering media bias and government overreach.
- **Chatrooms**: Multi-room chat system with restricted access for secure coordination.
- **Action Polls**: Reusable poll component to gauge citizen consensus on key issues.
- **Authentication**: Secure Google OAuth integration powered by Supabase.

## Tech Stack

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Backend/Auth**: Supabase (via `@supabase/ssr`)

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`.
3.  Copy `.env.local.example` to `.env.local` and add your Supabase credentials.
4.  Run the development server: `npm run dev`.
